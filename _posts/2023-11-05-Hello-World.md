코딩 공부를 시작하면 제일 먼저 C를 접하게된다.
그 중에서도 출력창에 `Hello World`가 국룰이지!
그래서 오늘은 `Hello World`를 출력하는 코드를 알아보도록 하자.

```C
#include <stdio.h>

int main() {

    printf("Hello World!!!\r\n");
    return 0;
}
```

이렇게 간단한 코드로 `Hello World`를 출력할 수 있다
